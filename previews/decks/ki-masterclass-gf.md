## Demo 1 — Vom Sprachmemo zur Markt-Frage

### Use Case
"Wir denken über ein neues Angebot oder einen neuen Markt nach — wie steht das Feld?"

### Features
**Voice Input** · **Context (Files & Bilder)** · **Deep Research**

### Live
Strategie-Gedanke als Voice Note einsprechen → KI als Sparringpartner mit Rückfragen → PDF oder Screenshot droppen → Konversation stößt an Wissensgrenze → Deep Research startet, läuft im Hintergrund.

===

<!-- @type: content -->
## Demo 2 — Calls vor- und nachbereiten

### Use Case
"Kundencall in einer Stunde — kein Kopf zum Mitschreiben."

### Features
**Voice Input** · **Connector Wissensdatenbank** · **Projects**

### Live
Vor dem Call zieht der Connector Kunden-Kontext aus der Wissensdatenbank → Briefing entsteht → nach dem Call Voice-Note als Debrief → KI strukturiert in Zusammenfassung, Entscheidungen, nächste Schritte → Connector schreibt zurück.

===

<!-- @type: content -->
## Demo 3 — Der automatische Morgen-Brief

### Use Case
"Den Tag starten ohne 20 Minuten Tool-Hopping."

### Features
**Connectors** · **Skills**

### Live
Connectors mit Kalender und Tasks verbinden → Fibery lesen → Skill `/morning-brief` aufrufen → konsolidierter Tagesdigest.

===

<!-- @type: content -->
## Demo 4 — Prozess rekonstruieren und konservieren

### Use Case
"Jahresprozess, vergessener Ablauf, nächste Woche fällig."

### Features
**Cowork** · **File Drop / Connector** · **Skill Creator**

### Live
Cowork bekommt letztjährigen Buchhaltungs-Folder → rekonstruiert Prozess aus Input und Output → erstellt Schritt-für-Schritt-Todo → wir gehen Teile live durch, Todo aktualisiert sich → Skill-Creator macht daraus eine wiederverwendbare Skill.
